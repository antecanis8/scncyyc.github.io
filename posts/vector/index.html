Vector

Apr 4, 2023

构造函数

构造函数声明 接口说明
vector() 无参构造
vector( size_type n, const t& t ) 构造并初始化n个val，即将前n个变量初始化为t
vector(const vector& x); 拷贝构造
vector (begin,end); 复制[begin,end)区间（左边包含，右边不包含）内某一个数组或vector的元素到vector中 
	vector<int> vec1(10,5);//前十个变量为5
	vector<int> vec2(vec1);
	int a[5]={1,2,3,4,5};
	vector<int> vec3(a,a+5);//将a数组的元素（从a[0]到a[4]，即左边包含右边不包含）用来初始化vector向量
	vector<int> vec4(vec1.begin(),vec1.begin()+5);//将vec1的元素（从vec1[0]到vec1[4]）用来初始化vector向量
	vector<int> vec5(&a[1],&a[4]);//将a[1]-a[3]范围内的元素用来初始化vector向量

容量操作

函数名称 功能说明
size_t size() const; 返回vec中元素的个数
size_t capacity() const 返回为vec分配的空间总大小
bool empty() const; 检测vec是否为空，是返回true，否则返回false
void clear(); 清空vec中所有元素
void push_back (const value_type& val); 在vec的最后一个向量后插入一个元素，其值为val，当且仅当插入前的size()==capacity()时，才会扩充空间
void pop_back(); 删除vec的最后一个元素
void assign (size_t n, const value_type& val); 将vec替换成n个val，注意是替换，不是在末尾添加
template void assign (InputIterator first, InputIterator last); 将[first,last)区间内的某个数组或vector的元素赋给vec
iterator erase (iterator position); 删除position位置的元素，例如vec.erase(vec.begin());
iterator erase (iterator first, iterator last); 删除[first,last)区间的元素，例如vec.erase(vec.begin(),vec.begin()+3);
iterator insert (iterator position, const value_type& val); 在vec的position元素（从第0个算起）的位置插入数值5，如vec为1,2,3,4，position为begin()+1，val为5，插入元素后为1,5,2,3,4
void insert (iterator position, size_t n, const value_type& val); 在vec的position元素（从第0个算起）的位置插入n个数，其值都为val
template void insert (iterator position, InputIterator first, InputIterator last); 在vec的position元素（从第0个算起）的位置插入[first,last)直接的元素
void swap (vector& x); 将vec与x中的元素整体交换

遍历方法

使用索引

for (int i = 0; i < vec.size(); i++) 
{
	cout << vec[i] << endl;
}

使用迭代器

for(auto itr = vec.begin(); itr != vec.end(); itr++)
{
	cout << *itr << endl;
}

不使用auto

for(vector<int>::iterator itr = vec.begin(); itr != vec.end(); itr++)
{
	cout << *itr << endl;
}

使用for的冒号表达式

for (auto element : vec)
{
	cout << element << endl;
}
for (auto &element : vec)
{
	cout << element << endl;
}
